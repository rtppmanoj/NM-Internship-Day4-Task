# Library Management System (MERN Stack)

A full-stack application for managing library books with full CRUD functionality.

## Features
- **Create**: Add new books via a React form.
- **Read**: View a list of all books fetched from MongoDB.
- **Update**: Edit book details.
- **Delete**: Remove books from the collection.

## Tech Stack
- **Frontend**: React.js, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
